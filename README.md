`Comic Collector`
```
Svrha: 
  Projekat iz predmeta "Mobilni sistemi i servisi".
Opis: 
  Na različitim mestima u gradu postavljeni su delovi stripova. Korisnici sakupljaju stripove prolaskom preko označenih mesta. 
  Svaki korisnik poseduje nalog uz pomoć kog može sakupljati ali i  kreirati i postavljati nove stripove. 
  Korisnici mogu postati "prijatelji" nakon čega imaju mogućnost deljenja sakupljenih delova stripova ukoliko se nalaze na određenoj      udaljenosti jedan od drugoga.
 ```
